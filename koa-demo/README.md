A collection of simple demos of [Koa](http://koajs.com/), a web application framework for Node.

![Logo](logo.png)

## How to use

First of all, check your Node version.

```bash
$ node -v
v8.0.0
```

```bash
$ cd koa-demos
$ npm install
```
